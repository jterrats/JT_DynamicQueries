Installation Steps

# Clone repository
git clone https://github.com/jterrats/JT_DynamicQueries.git
cd JT_DynamicQueries

# Install dependencies
npm install

# Deploy to your org
npm run deploy:named-credential

# Run tests (optional)
npm test
